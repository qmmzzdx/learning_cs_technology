## 计算机网络知识体系

### 1. 网络基础与协议栈
- **网络模型**：OSI七层模型 vs TCP/IP四层模型
- **协议基础**：封装与分用、报文格式、寻址机制
- **数据链路层**：MAC地址、ARP协议、以太网帧
- **网络层**：IP协议、ICMP、路由协议、子网划分

### 2. TCP/IP 深度解析
- **TCP协议**：三次握手、四次挥手、可靠传输机制
- **UDP协议**：无连接通信、适用场景、性能优势
- **拥塞控制**：慢启动、拥塞避免、快速重传/恢复
- **流量控制**：滑动窗口、零窗口探测、Nagle算法
- **性能优化**：TCP参数调优、BBR算法、缓冲区管理

### 3. HTTP/HTTPS 与 Web 协议
- **HTTP协议**：请求/响应模型、方法定义、状态码
- **HTTP进阶**：持久连接、管线化、缓存机制
- **HTTPS安全**：TLS握手、证书体系、混合加密
- **HTTP/2 & HTTP/3**：二进制分帧、多路复用、QUIC协议
- **WebSocket**：全双工通信、握手过程、实时应用

### 4. 网络编程与实践
- **Socket编程**：TCP/UDP套接字、IO多路复用
- **高性能网络**：Reactor模式、Proactor模式、连接池
- **Linux网络栈**：收发包流程、协议栈优化、eBPF应用
- **RPC框架**：gRPC、Thrift、序列化协议、服务发现

### 5. 网络架构与中间件
- **负载均衡**：LVS、Nginx、算法策略、健康检查
- **CDN原理**：缓存策略、调度算法、边缘计算
- **DNS系统**：域名解析、负载均衡、高可用设计
- **API网关**：路由转发、限流降级、认证授权

### 6. 网络安全与运维
- **网络攻击**：DDoS防护、中间人攻击、DNS劫持
- **安全协议**：VPN、IPSec、WireGuard
- **网络诊断**：tcpdump、Wireshark、ping/traceroute
- **监控告警**：网络流量、连接数、延迟指标

### 7. 云原生网络
- **容器网络**：Docker网络模式、CNI标准、Calico/Flannel
- **服务网格**：Istio、Envoy、mTLS、流量管理
- **云网络**：VPC、安全组、负载均衡器、NAT网关
- **网络策略**：网络策略、微隔离、零信任架构
